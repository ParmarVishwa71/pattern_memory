# 🧠 Smart Pattern Memory Game

A modern and interactive browser-based memory game developed using HTML, CSS, and Vanilla JavaScript. Players improve their memory by remembering and repeating color patterns while progressing through multiple difficulty levels.

## 📌 Project Description

Smart Pattern Memory Game is a fun and challenging memory game that tests players' concentration and pattern recognition skills. The game includes multiple difficulty levels, score tracking, statistics, timer, keyboard accessibility, responsive design, and smooth animations for an engaging user experience.

## 🚀 Installation & How to Run

1. Download or Clone the repository.

```
git clone https://github.com/your-username/smart-pattern-memory-game.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

OR

Use VS Code with **Live Server**.

---

## 🎮 How to Play

1. Click **Start Game**.
2. Watch the color pattern carefully.
3. Repeat the same pattern.
4. Complete each level to earn points.
5. Try to beat your highest score.
